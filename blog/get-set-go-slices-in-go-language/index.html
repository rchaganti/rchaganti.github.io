In the previous part of this series, you learned about arrays in Go language. As you have learned, arrays have a fixed size and therefore you cannot resize (grow or shrink) arrays. This limitation can be overcome using slices in Go language. In this part, you will learn about slices and how to use this data type in your Go programs.

Slices

A slice in Go language is a slice of an underlying array! Yes, no pun intended. ðŸ˜Š

Here is how the Go language specification describes slices. A slice is a descriptor for a contiguous segment of an underlying array and provides access to a numbered sequence of elements from that array.

First, let us look at slicing arrays.

Slicing arrays

Take a look at this example.

package main

import "fmt"

func main() {
	var languages = [6]string{"Go", "Python", "Rust", "Java", "C#", "PowerShell"}
	fmt.Println("Elements in the string array are", languages)

	var langSlice = languages[0:2]
	fmt.Println("Elements in the derived slice are", langSlice)
}

In the above example, the languages array has six elements. A slice of these elements – from index 0 to 2 – is assigned to the variable langSlice. This variable is of slice data type. In the variable assignment, you have specified the the slice as start_index:end_index in square brackets. In this specification, the both start_index and end_index are optional. So, for example, if you specify [:3], the first four elements from the array. If you specify [4:], elements from index 4 till the end of the array get assigned to the slice. Finally, skipping both – [:] – will simply return all elements from the array.

Declaring a slice type

The declaration of a slice is no different from an array declaration.

var <slice-name>[]<elements-type>

The only difference, if you have observed is that there is no size specification. You just leave out the element count in square brackets. Similar to an array, the number of element in a slice denotes the length of the slice and you can use the built-in len function to determine the length of the slice. Here is a quick example.

package main

import "fmt"

func main() {
	languages := [2]string{"Go", "Python"}
	fmt.Println("values in the slice are", languages)

	// get length of the slice
	var l = len(languages)

	fmt.Println("Length of the slice is", l)
}
