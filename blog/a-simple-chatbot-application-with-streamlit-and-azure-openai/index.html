With GenAI gaining momentum, several AI developers and companies are seeking ways to build and deploy user-friendly AI applications that are easy to build and deploy. Developing enterprise-grade AI applications certainly requires a good amount of experience across different layers of the application stack. I am not a front-end or full-stack developer. However, when presenting my ideas, it is useful to have a user interface to demonstrate the functionality. To this extent, I started using Streamlit. Streamlit is an open-source app framework that makes it easy to create web applications. In this article, we will use Streamlit to create a simple chatbot application and use Azure OpenAI for all LLM interactions.

If you are new to Azure OpenAI services and their API, you can get a quick start by reading a series of articles I am writing on this topic.
