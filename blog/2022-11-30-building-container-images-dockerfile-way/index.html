Building container images - using no tools

In the last part of this series, we looked at building container images with just built-in OS tools. While that was fun and good to know the details behind building a container image, you don’t need to do that. With the introduction of OCI image specification in 2015, there are many tools now that you can use to build container images. In today’s article, we will review usage of Dockerfile for building container images.

Dockerfile introduction

When you think of building container images, the first thing that comes to mind is Dockerfile. A Dockerfile is a text document with a series of instructions around how the image needs to be built.
